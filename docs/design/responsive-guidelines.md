# Distill - Responsive Guidelines

## ブレークポイント

| 名前 | サイズ | ターゲット |
|------|--------|----------|
| sm | < 640px | スマートフォン |
| md | 640px〜1024px | タブレット |
| lg | > 1024px | デスクトップ（メイン） |

## レイアウト変化

### Desktop (> 1024px) — Primary

```
max-width: 1200px, 中央寄せ
padding: 32px

┌─────────────────────────────────┐
│  Distill              [Toggle]  │
│                                 │
│  [URL入力________________] [▶]  │
│                                 │
│  [🖼] タイトル                   │
│       チャンネル | 再生時間      │
│                                 │
│  [全文][要約][ブログ][SNS] [PDF] │
│  ┌─────────────────────────┐    │
│  │ コンテンツ               │    │
│  └─────────────────────────┘    │
└─────────────────────────────────┘
```

### Tablet (640px〜1024px)

```
padding: 24px

- URL入力とボタンは横並びを維持
- メタ情報のサムネイルは非表示
- コンテンツ幅はフル
```

### Mobile (< 640px)

```
padding: 16px

┌───────────────────┐
│ Distill   [Toggle]│
│                   │
│ [URL入力________] │
│ [Distill ▶ フル幅]│
│                   │
│ タイトル           │
│ チャンネル         │
│                   │
│ [全文][要約][ ▶ ] │← タブはスクロール可能
│ ┌───────────────┐ │
│ │ コンテンツ     │ │
│ └───────────────┘ │
│                   │
│ [📥 PDF フル幅]   │
└───────────────────┘
```

## コンポーネント別対応

| コンポーネント | Desktop | Mobile |
|-------------|---------|--------|
| URL入力 + ボタン | 横並び | 縦積み |
| タブバー | 全タブ表示 | 横スクロール |
| PDFボタン | タブバー右端 | フッター固定 |
| メタ情報 | サムネ + テキスト | テキストのみ |
| コンテンツ | padding 24px | padding 16px |

## タッチ対応

- タッチターゲット: 最低44px
- タブはスワイプで切替可能
- PDFボタンはモバイルではフローティング
