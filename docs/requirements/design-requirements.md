# Distill - デザイン要件

## 美的方向性

### トーン & ムード

| 項目 | 値 |
|------|-----|
| トーン | Minimal-Industrial |
| ムード | 知的・クリーン・信頼感 |
| コンセプト | 蒸留所のような洗練された空間 |
| 差別化 | ダーク基調 + アクセントカラーの蒸留器イメージ |

### ビジュアルキーワード

- 蒸留（Distillation）: 不要なものを取り除き、本質だけを残す
- ミニマル: 余白を活かしたクリーンなレイアウト
- インダストリアル: 機能美、素材感
- プロフェッショナル: 信頼感のあるツール感

### 参照イメージ

- 蒸留所の銅器のような温かみのあるアクセント
- ラボラトリーのような整然としたレイアウト
- ターミナル/IDEのような機能性

## UIコンポーネント方針

### ライブラリ

| カテゴリ | 技術 |
|---------|------|
| コンポーネント | shadcn/ui |
| スタイリング | Tailwind CSS 4 |
| アニメーション | motion/react |
| アイコン | Lucide React |
| ユーティリティ | cn() (clsx + tailwind-merge) |

### コンポーネントスタイル方針

- ボーダーは細く（1px）、低コントラスト
- 角丸は控えめ（4px〜8px）
- シャドウは最小限（ダークモードではほぼ不使用）
- ホバー時は subtle な brightness 変化
- フォーカスリングは Primary カラー

### アニメーション方針

| 項目 | 値 |
|------|-----|
| 基本duration | 150ms〜200ms |
| イージング | ease-out |
| 使用プロパティ | transform, opacity のみ |
| タブ切替 | フェードイン（150ms） |
| プログレス | スムーズなバー進行 |
| 結果表示 | stagger フェードイン |

**禁止**: レイアウトプロパティのアニメーション、過剰なバウンス

## レスポンシブ方針

### ブレークポイント

| 名前 | サイズ | レイアウト |
|------|--------|----------|
| mobile | < 640px | スタック、フルワイド |
| tablet | 640px〜1024px | パディング調整 |
| desktop | > 1024px | 最大幅1200px、中央寄せ |

### モバイルファースト

- PC優先設計だが、モバイルでも最低限使える
- タブはスクロール可能に
- PDF保存ボタンは常にアクセス可能

## Anti-Patterns（禁止事項）

| カテゴリ | 禁止パターン | 理由 |
|---------|------------|------|
| フォント | Inter, Roboto, Arial | AI slop |
| カラー | 紫グラデーション on 白背景 | AI slop |
| レイアウト | 予測可能なSaaSテンプレート感 | 差別化不足 |
| エフェクト | 過剰なドロップシャドウ | AI slop |
| エフェクト | グロー効果 | AI slop |
| エフェクト | グラデーション背景 | 安っぽさ |
| UI | 丸すぎる角丸（16px+） | カジュアルすぎる |
